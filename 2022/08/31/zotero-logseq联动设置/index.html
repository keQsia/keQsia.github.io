zotero-logseq 联动设置

  1. 首先，需要在 zotero 官网上生成对应软件的私钥，网址为：https://www.zotero.org/settings/keys。一般在 Personal Library 下勾选前两项即可，选择 Save Key 即可生成私钥并复制。

  2. 对 logseq 进行如下设置：

    在 Zotero API key 中填写刚才生成的私钥，在 User or Group ID 中填写官网上面只有附件文件夹和条目文件夹对应设置填写正确才能通过 /zotero 命令直接插入文献。
