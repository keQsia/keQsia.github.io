GitHub 配置 SSH Key

  1. 找到本地用户文件夹下 .ssh 文件夹中的 id_rsa.pub 文件，复制其中的内容；
  2. 登录 github，点击头像选择 Settings；
  3. 选择 SSH and GPG keys 这一项，点击 New SSH key 按钮，Tile 上键入便于记忆的名字，将复制的内容粘贴到 Key 中，选择 Add SSH Key，这样就完成了公钥的配置；
  4. 可以在博客文件夹上右键 git bash here，打开 bash 窗口，键入 ssh -T git@github.com，假如提示成功信息即证明配置成功。

参照网上ssh建立互信办法，生成pug文件和在github上建立公钥，注意设置密码为空
